Sambungkan ke Google Drive
Data otomatis ter-sync setiap web app dibuka. Tim cukup replace file Excel di Drive.
Cara Setup (sekali saja)
1
Upload file Aging_AR.xlsx ke Google Drive (folder tim)
2
Buka script.google.com → New project → paste isi file apps-script.gs → ganti FILE_NAME
3
Klik Deploy → New deployment → Web app · Execute as: Me · Access: Anyone → Deploy → copy URL
4
Paste URL di kolom atas → klik Sambungkan. Atau isi langsung di konstanta APPS_SCRIPT_URL di file HTML → upload ulang ke GitHub → semua device otomatis connect tanpa perlu input apapun.
